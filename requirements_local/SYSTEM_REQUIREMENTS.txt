Project LFR System Requirements:

1. Windows XP: 512MB RAM minimum, 1GB recommended (SHC_core package required).

2. Windows Vista / 7: 1GB RAM minimum for Vista, 2GB for Windows 7.

3. Windows 8 / 8.1: 2GB RAM minimum. Transparency patches might be needed.

4. Windows 10 / 11 (Home, Pro, Enterprise): 4GB RAM minimum. OS can take up to 25% of resources for Bloatware.

Optimization: It is highly recommended to remove Cortana, telemetry, and third-party "optimizers" (IObit, Alcohol 120%) for stable performance.
